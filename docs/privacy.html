Privacy Policy

Effective Date: March 31, 2026

1. Introduction

This privacy policy describes how the BASG Corp Invoice Payment Tool ("Tool") handles data. This Tool is an internal application used exclusively by BASG Corp.

2. Data We Access

The Tool accesses the following data from QuickBooks Online via Intuit's API:

  • Invoice numbers, amounts, and statuses
  • Customer names associated with invoices
  • Payment method information

3. How We Use Data

Data is accessed solely to record payments against invoices in BASG Corp's QuickBooks Online account. Data is used in real time and is not stored outside of QuickBooks Online, except for OAuth tokens required to maintain the API connection.

4. Data Sharing

We do not sell, share, or distribute any QuickBooks Online data to third parties. Data remains exclusively within BASG Corp's systems and QuickBooks Online account.

5. Data Security

OAuth 2.0 credentials are stored securely in environment variables on BASG Corp's private systems. Access tokens are refreshed automatically and are never exposed publicly.

6. Data Retention

The Tool does not maintain a separate database. All accounting data resides in QuickBooks Online. OAuth refresh tokens are retained only as long as needed to maintain the API connection.

7. Contact

For privacy-related questions, contact BASG Corp management.
